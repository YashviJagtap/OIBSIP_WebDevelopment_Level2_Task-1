# 🌟 Neon Calculator

A **creative and interactive calculator** built using **HTML, CSS, and JavaScript**.  
It performs basic mathematical operations like **addition, subtraction, multiplication, division**, and supports **decimal values**.

## **Features**

- **Stylish Neon Theme** with glowing buttons and text.
- **Interactive Hover Effects**: Buttons scale and glow when hovered.
- **Responsive Grid Layout**: Perfectly aligned buttons for easy use.
- **Basic Arithmetic Operations**: +, −, ×, ÷
- **Decimal Support**
- **Clear Button** to reset calculations.
- **Error Handling**: Division by zero shows `"Error"`.

---

## **Technologies Used**

- **HTML5** – Structure of the calculator.
- **CSS3** – Styling with grid, animations, hover effects, neon glow.
- **JavaScript** – Logic for calculations, event handling, and dynamic updates.
